# iPhoneで「Spoon定期スタジオ」を使う手順(担当: コトハ)

## アプリのURL
**https://seizmiggy.github.io/spoon-teiki-studio/**

## ホーム画面に追加する(最初に1回だけ)
1. iPhoneの **Safari** で上のURLを開く(ChromeではなくSafariで!)
2. 画面下の **共有ボタン(□に↑のマーク)** をタップ
3. メニューを下にスクロールして **「ホーム画面に追加」** をタップ
4. 名前が「定期スタジオ」になっているのを確認して **「追加」**

これでホーム画面に紫の「定期」アイコンが増えます。次からはこのアイコンをタップするだけ。アプリのように全画面で開き、**電波がない場所でも動きます**。

## 使い方のポイント(iPhone版)
- **道具箱**: 画面下の「▲ 道具箱をひらく」をタップすると設定パネルが出ます。ふだんは閉じたままでOK
- **文字の配置**: マス目の空きをタップ → 入力
- **移動**: 文字をドラッグ、または選択して道具箱の ◀▲▼▶ ボタンで1マスずつ
- **複数選択**: 文字を **0.5秒長押し** すると選択に追加されます(もう一度長押しで解除)
- **Spoonに貼る**: 「コピー」ボタン → Spoonアプリのプロフィールコメント欄に貼り付け

## 更新について
アプリを新しくしたときは、開き直すと自動で最新版になります(反映まで少し時間がかかることがあります。おかしいときは一度閉じて開き直してください)。

## 困ったときは
- コピーできない → 自動で「手動コピー」の画面が出ます。文字を長押し→すべてを選択→コピー
- 表示が崩れた → docs/ユズ_実機検証チェックリスト.md の項目を確認して報告してください
