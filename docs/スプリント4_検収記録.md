# スプリント4 検収記録(記録者: ミナト / 2026-07-19)

## 発注者リクエスト(起点)
画像→点字変換の時点で、90%/80%/70%/60%/50%のいずれかを指定してから変換したい。
「100%」の基準は発注者に確認: **200字予算内で入る最大サイズ**(推奨案どおり)。

## 実装(レン)
- 変換モーダルに「縮小率」プルダウンを追加(100/90/80/70/60/50%、既定100%)。
- `convAutofit()` が求める予算内最大の幅(点字文字数)を `conv.baseCols` として保持し、それを100%の基準に固定。
- `applyConvScale()` が選択中の%を `baseCols` に掛けて「幅(点字文字数)」スライダーへ反映(下限4・上限44でクランプ、既存の幅ロックによる上限キャップも従来どおり適用)。
- 道具箱(変換モーダル)を開き直すたびに縮小率は100%から選び直せる(前回値を引きずらない)。
- 実装は既存の「予算に自動フィット」の仕組みをそのまま利用しており、画質劣化は伴わない(元画像から都度再サンプリングするため)。
